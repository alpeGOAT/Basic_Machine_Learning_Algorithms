# Interquartile Range (IQR) Outlier Detection
This project uses quartiles to identify outliers in a small numerical dataset. Q1, Q2, and Q3 are calculated, and the interquartile range is found using Q3 minus Q1. Values below Q1 - 1.5 × IQR or above Q3 + 1.5 × IQR are classified as outliers. 
In the example, 50 is detected as an outlier and displayed separately in a box plot.
