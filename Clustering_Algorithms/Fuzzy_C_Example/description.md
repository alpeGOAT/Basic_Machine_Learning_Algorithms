# Fuzzy C-Means with a Sample Dataset
This project demonstrates soft clustering using the Fuzzy C-Means algorithm. Each data point receives a membership score between 0 and 1 for every cluster instead of belonging completely to only one group. 
The project creates 100 two-dimensional points, calculates three cluster centers, and updates membership values until the algorithm becomes stable. The final visualization shows both fuzzy membership levels and the strongest hard cluster of each point.
