# K-Means Clustering with a Sample Dataset
This project implements the K-Means algorithm on an artificial dataset containing 500 two-dimensional data points and three natural groups. 
The data is standardized before clustering. Random centroids are created, each point is assigned to its nearest centroid using Euclidean distance, and the centroid positions are updated by calculating the mean of their assigned points. 
The final graph shows the three discovered clusters and their stable centroids.
