# DBSCAN with a Sample Dataset
This project demonstrates density-based clustering using an artificial dataset with 300 points and four groups. DBSCAN forms clusters according to the eps radius and minimum number of neighboring points. 
Dense points become core points, less dense points may become border points, and isolated points are marked as noise with the label -1. The result is visualized, and clustering quality is evaluated using metrics such as the Silhouette Coefficient.
